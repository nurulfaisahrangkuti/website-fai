HTML dan CSS

HTML (Hyper Text Markup Language) adalah bahasa markup yang digunakan untuk membuat Halaman web. HTML adalah markup yang digunakan untuk mendefinisikan struktur halaman web. Hal ini seperti misalnya membuat elemen paragraf, gambar, link dan lain sebagainya. HTML tersusun atas elemen/tag HTML. Sedangkan CSS (Cascading Style Sheet) adalah bahasa yang digunakan untuk mengatur tampilan (gaya) elemen-elemen HTML. Dengan CSS, kita dapat memberikan warna background, border, mengatur posisi elemen, ukuran font dan lain sebagainya.

Fungsi Html

Fungsi HTML

  • Sebagai struktur dasar pembuatan halaman web.
  • Memberikan makna kepada konten web dengan menggunakan tag.
  • Mengintegrasikan berbagai jenis media seperti audio, video, gambar dan lain sebagainya.

Fungsi CSS

  • Mengontrol aspek visual halaman web.
  • Membuat halaman web menjadi responsif.
  • Mempermudah dalam mengelola dan memperbaharui desain dengan memisahkan tampilan dan konten.
